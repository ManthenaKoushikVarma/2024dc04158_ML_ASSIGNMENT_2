Problem Statement: The goal is to predict a student's final academic outcome (FinalGrade) based on their demographic information, study habits (e.g., study hours, attendance), and psychological factors (e.g., stress levels, motivation). This is a classification task aimed at identifying students who may need additional academic support.

Dataset Description:

Source: Kaggle (Student Performance and Learning Behavior Dataset).

Instance Size: 14,003 student records.

Feature Size: 16 attributes including StudyHours, Attendance, StressLevel, Motivation, InternetAccess, and LearningStyle.

Target Variable: FinalGrade (Categorical/Ordinal).
